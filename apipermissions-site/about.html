About Microsoft API Permissions Explorer

Project Overview

The Microsoft API Permissions Explorer is a comprehensive, user-friendly tool for developers, administrators, and security professionals to explore and understand the various permissions available in Microsoft's API ecosystem, particularly in Microsoft Graph API.

This site provides up-to-date information about available permissions, their descriptions, and how they function within the Microsoft identity platform.

Data Sources

All permission data on this site is sourced directly from the following Microsoft repositories:

  • permissions-descriptions.json - Main permissions metadata
  • ProvisioningInfo.json - Information about permission provisioning
  • permissions.json - Additional permission details

The site automatically updates when changes are made to these sources, ensuring that the information is always current.

How to Use This Site

The Microsoft API Permissions Explorer provides several ways to explore and understand API permissions:

  • Search: Use the search bar to find specific permissions by name or description.
  • Browse: Explore the complete list of permissions on the home page.
  • Detail View: Click on "Details" for any permission to see comprehensive information.
